Upon opening HotSpots, a map of Macalester's campus will be displayed for the user. The map contains pins with icons that represent what the location is. When a user clicks on an icon, a window appears that contains the name of the location and a short description. In the upper right corner, there is a filter that can remove icons from the map. In the lower right corner, there is a button that will recenter the map to Macalester's campus.
