Upon opening HotSpots, a map of Macalester's campus will be displayed for the user. The map contains pins with icons that represent what the location is. When a user clicks on a pin, a window appears that contains the name of the location, a short description, and tags that tell the user what students do at this location. In the upper right corner, there is a filter that can remove icons from the map based on tags or type. You can suggest your own pins to be added to the map by clicking the suggestion tab on the bottom. You can then add the pin from the community tab. In the lower right corner, there is a button that will recenter the map to Macalester's campus.
